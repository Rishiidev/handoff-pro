# Contributing to Handoff Pro

Thanks for considering a contribution. This is a small, focused skill — keep PRs scoped accordingly.

## What's welcome

- Bug reports — if SAVE or LOAD misbehaves in Claude.ai, Cowork, or Claude Code, open an issue with the environment and what you expected vs. got.
- Improvements to the 12 captured elements in `SKILL.md` or the template in `references/handoff-template.md`.
- Additional storage-path support for other environments.
- Clearer install/usage docs.

## What's not a fit

- General-purpose "memory" or "notes" features unrelated to session handoff — keep this skill focused on one job.
- Breaking changes to the document format without a migration note.

## How to contribute

1. Fork the repo.
2. Make your change in a branch.
3. Open a PR describing what changed and why.
4. Keep `SKILL.md` and `references/handoff-template.md` in sync if you touch the captured-fields list.

## Questions

Open an issue — happy to discuss before you write code.
